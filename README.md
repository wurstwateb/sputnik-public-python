# Sputnik Public

Публичный локальный модератор ответов Jackbox через Selenium.

В этой версии нет Telegram, Discord, TikTok, ИИ API, токенов, вебхуков,
отправки данных и телеметрии. Программа читает страницу модератора в вашем
браузере, проверяет ответы по локальным спискам и нажимает Accept/Reject.

## Запуск

1. Установите Python 3.11+ и Edge либо Firefox.
2. Дважды нажмите `START.bat` — это единственный запускающий скрипт.
3. Если Python отсутствует, `START.bat` установит его через `winget`.
4. При первом запуске он создаст `.venv`, установит Selenium и проверит приложение.
5. В окне выберите браузер и нажмите «Запустить модерацию».
6. В открывшемся браузере войдите в комнату Jackbox.

Selenium Manager сам подбирает драйвер браузера. Не запускайте geckodriver или
msedgedriver вручную и не настраивайте порты.

## Фильтры

- `blacklist.txt` — слова и фрагменты;
- `blacklist_names.txt` — запрещённые ники;
- `blacklist_regex.txt` — регулярные выражения Python.

Строки с `#` в начале и пустые строки игнорируются. После изменения списков
перезапустите модерацию.

## Рисунки

В публичной версии нет внешней ИИ-проверки. В настройках можно выбрать единое
правило: принимать или отклонять рисунки.

Другие `.bat` и `.ps1` для запуска не нужны и намеренно не включены.
